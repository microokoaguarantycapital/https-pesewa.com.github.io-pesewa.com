# Pesewa.com - Progressive Web App

## Overview
Pesewa.com is a revolutionary, trust-based peer-to-peer emergency micro-lending platform that connects borrowers and lenders through trusted referral networks. This PWA provides a seamless, installable, and offline-capable experience across all devices.

## Features
- **Trust-Based Lending**: Join via referral only, with dual referrer verification
- **Multiple Tiers**: Basic (≤1,500/week), Premium (≤5,000/week), Super (≤20,000/week)
- **10 Emergency Categories**: Transportation, Data, Food, Medicine, Utilities, etc.
- **Real-Time Updates**: Live loan tracking and notifications
- **Offline Support**: Request loans and view data even without internet
- **Cross-Platform**: Installable on desktop and mobile devices
- **Global Support**: 10+ countries with local currency support
